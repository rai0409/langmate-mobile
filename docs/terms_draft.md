## Service

LangMate provides profile discovery, reciprocal matching, chat, and basic
safety actions for language exchange. Learning support tools in the current product preview
are mock previews only and do not provide real AI translation, correction, or
reply generation.

## User Responsibilities

Users should:

- Provide accurate account and profile information.
- Respect other users and communicate safely.
- Use report and block features for safety issues.
- Avoid sharing highly sensitive personal information in profiles or chat.
- Follow applicable laws and platform rules.

## Prohibited Conduct

The production terms should prohibit harassment, threats, hate or abusive
content, sexual exploitation, spam, scams, impersonation, unauthorized scraping,
attempts to bypass security rules, and illegal activity.

## Content and Moderation

Users remain responsible for content they submit. LangMate may need to review
reports, restrict accounts, preserve safety records, or remove content through a
future admin moderation workflow. Current repository work defines only a
local/admin data model baseline; it is not a production moderation service.

## Account Deletion and Termination

Users should be able to request account deletion. The current app baseline can
store an account deletion request record, but actual deletion requires a future
server-side processor and retention review. Some records may need to be
retained, anonymized, or delayed where required for safety, abuse prevention,
legal obligations, or dispute handling.

## Disclaimers To Review

Before publication, counsel should review service availability, limitation of
liability, user-generated content ownership/license, warranty disclaimers,
